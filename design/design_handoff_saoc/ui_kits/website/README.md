# SAOC Website UI Kit

High-fidelity recreation of the SAOC website's component vocabulary — mark + wordmark, palette, type, button system, eyebrow pills, navigation blocks, dark "next show" band, event rows, society cards, page hero, footer.

`index.html` is a one-page demo showing every component in context. Components live in `components.jsx` and are exported to `window` so other JSX files can reuse them.

The full multi-page prototype (Home / About / Societies / Judging / National Show / Events / Contact) lives in `../../SAOC Website.html`. This kit is the simpler, _modular_ version designed to be lifted into other SAOC surfaces (donor packs, exhibitor flyers, etc.).
